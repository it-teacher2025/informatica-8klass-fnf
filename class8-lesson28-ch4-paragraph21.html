Тема 4: «Опрацювання мультимедійних об’єктів». Урок № 28. (До § 21)

Програмне забезпечення для опрацювання відео

Інтерактивний план-конспект уроку для 8 класу НУШ

🎯 Мета уроку

  • Освітня: ознайомити учнів з поняттям програмного забезпечення для опрацювання відео, його видами та можливостями.
  • Розвивальна: розвивати навички роботи з програмами для захоплення та конвертування відео, критичне мислення та креативність.
  • Виховна: виховувати відповідальне ставлення до використання інформаційних технологій, уважність та дисциплінованість.

✅ Очікувані результати

Після уроку учні зможуть:

  • розрізняти види програм для опрацювання відео.
  • називати приклади програм для захоплення та конвертування відео.
  • виконувати базові операції із захоплення відео за допомогою OBS Studio.
  • конвертувати відеофайли за допомогою HandBrake.

📊 Групи результатів (НУШ)

Візуалізація важливості груп результатів для цього уроку.

Учень / учениця:

  • ГР1 – працює з інформацією, даними, моделями;

  • ГР2 – створює інформаційні продукти;

  • ГР3 – працює в цифровому середовищі;

  • ГР4 – безпечно та відповідально працює з інформаційними технологіями.

📚 Тип уроку

Урок вивчення нового матеріалу з елементами практичної роботи.

💻 Обладнання та ПЗ

  • Персональні комп'ютери з операційною системою Windows або macOS.
  • Програми для захоплення відео: OBS Studio, Icecream Screen Recorder, CamStudio тощо.
  • Програми для конвертування відео: HandBrake.
  • Програми для перегляду відео: медіаплеєри, що підтримують різні формати.

🔗 Міжпредметні зв'язки

  • Мистецтво (створення відеороликів, монтаж).
  • Українська мова та література (створення відео для презентацій, відеопроєктів).
  • Технології (використання програм для обробки відео у проєктній діяльності).

Перевірка знань

Натисніть на картку, щоб побачити відповідь.

  • 1. Що таке захоплення відео?
    Процес перетворення відеосигналу із зовнішнього пристрою в цифрову форму з подальшим записом у відеофайл називають захопленням відео.
  • 2. З яких пристроїв можна захоплювати відео?
    З аналогового пристрою, цифрових відеокамер, екрана монітора.
  • 3. Які можливості надає програма OBS Studio?
    Дозволяє захоплювати одночасно відео з екрана та вебкамери, захоплювати звук із мікрофона та програм; довільно компонувати сцени, створювати декілька сцен і перемикатися між ними; здійснювати пряму трансляцію відео на популярні відеосервіси (YouTube, Instagram тощо).
  • 4. Якими програмами для перегляду відеофайлів ви користуєтеся?
    Опишіть, які можливості вони надають.
    Приклад: VLC Media Player.

    Можливості: відтворення більшості відео- та аудіоформатів, потокового відео, підтримка субтитрів, конвертація медіафайлів.
  • 5. Коли виникає потреба змінити формат відеофайлу?
    Коли файл має великі розміри, коли потрібно переглядати на різних пристроях, які підтримують різні формати, або коли потрібно оптимізувати файл для завантаження в Інтернет.
  • 6. Для чого призначені програми-конвертери?
    Програми-конвертери призначені для перекодування медіафайлів (відео- та/або аудіофайлів) з одного формату в інший.

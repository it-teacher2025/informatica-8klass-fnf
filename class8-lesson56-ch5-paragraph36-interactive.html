Тема 5: «Алгоритми та програми». Урок № 56. (До § 36)

Графічні методи модуля tkinter

Інтерактивний план-конспект уроку для 8 класу НУШ

Інтерактивний Canvas-редактор

Спробуйте малювати різні фігури на полотні, змінюючи їх властивості. Цей інтерактивний інструмент імітує роботу з `tkinter.Canvas`.

Завдання для самостійного виконання

  • Намалюйте складне зображення (наприклад, будинок, сонце, дерево), використовуючи різні графічні методи модуля `tkinter`.
  • Збережіть ваш код Python з `tkinter` для малювання.
  • Додайте можливість змінювати колір фігур за допомогою вводу користувача.
